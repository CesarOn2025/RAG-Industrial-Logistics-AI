# AI RAG System for Industrial Logistics

This project demonstrates a Retrieval-Augmented Generation (RAG) system that processes technical manuals and answers questions using AI.

## Features
- PDF processing (Blucher manuals)
- Semantic search using embeddings
- Vector database with ChromaDB
- AI-generated responses using GPT

## Technologies Used
- Python
- LangChain
- ChromaDB
- OpenAI GPT

## Project Goal
To improve efficiency in industrial operations by enabling fast and intelligent access to technical information.

The OpenAI API key has been removed for security reasons.
